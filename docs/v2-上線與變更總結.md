## 4. 轉盤規則（現行）

```
候選 = 營業中（含未知）
     ∩ 正餐 eligible（is_wheel_eligible）
     ∩ 側欄分類（有選時：所選 ∪ 其他）
     → 綜合分排序 → Top 40
```

**正餐 eligible（`categories.is_wheel_eligible`）**

- 排除分類：`下午茶咖啡廳類`、`其他`
- 排除店名含：自助冰、剉冰、刨冰、雪花冰、燒仙草、仙草冰、豆花、冰淇淋…

**仍會進轉盤的範例：** 六扇門（火鍋）、麥當勞（炸物）、非凡豆漿（便當）

**不進轉盤的範例：** 哈哈自助冰、燒仙草（冰品／下午茶）

側欄若**手動勾選「下午茶咖啡廳類」**，轉盤仍可依你的篩選顯示飲料／甜點店。

## 8. 已知限制

1. `price_level` 約 50% 為 null — 平價篩選預設「含未標價位」
2. 快取無 Google `specialDays` — 國定假日仍用平日營業時段
3. 轉盤 40 格在手機上對照表較長，需捲動
4. `places_cache.json` 僅本機；更新快取後需同步 `places_cache.public.json` 再 push
